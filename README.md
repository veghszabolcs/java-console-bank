## Java "banki" konzolos app

Java nyelven készült, Maven alapú banki alkalmazás JDBC-vel és MySQL adatbázissal. A projekt tartalmaz felhasználókezelést, bejelentkezést és egy rétegzett architektúrát (service, repository, model).

## Java "bank" console app

A Maven-based banking application built in Java, using JDBC and a MySQL database. The project includes user management, login functionality, and a layered architecture (service, repository, model).
